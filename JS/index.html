Javascript Fundamental
Section Task Description
Variable Variable (var, let, const) Variable for store data, Var and Let store data that can change later, but Const can't change later.
Scope (block, functional, global) Scope determines the accessibility of variable, block is limited to curly bracket, functional in function, and global can use througout the script.
Hoisting Hoisting is the process of moving variable and function declarations to the top of their containing scope.
Data Types Primitive Types String is a sentece with Quote. Number is a number. Boolean is a True or False. Undefined is a variable that has not been assigned a value. Null is a special value that represents the absence of an object. Symbol is a unique type of value, even the value is same.
Object Object is like an array, but its store data in key-value pair.
Typeof Operator Typeof Operator is use to determines the type of given objeck/variable/operant
Operators Assignment Assignment Operator is use to assign value to variable (= , +=, -=, *=, /=)
Comparison Comparison Operator is use to compare value (==, ===, !=, !==, >, <, >=, <=)
Arithmetic Arithmetic Operator is use to calculate value (+, -, *, /, %)
Bitwise Bitwise Operator is use to calculate value that return Binary (&, |, ^, ~, <<, >>)
Logical Logical Operator is use to calculate value with logic gate that return Boolean (&&, ||, !)
