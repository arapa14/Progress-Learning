HTML Tags, Attributes, and Descriptions

Task Description
<html> The root element of an HTML document. It wraps all the content in the document.
<head> Contains metadata, title, and links to stylesheets or scripts for the document.
<body> Defines the main content of the HTML document visible to users.
<title> Sets the title of the web page displayed on the browser tab.
<h1> to <h6> Defines headings, with <h1> being the largest and <h6> the smallest.
<p> Defines a paragraph in the document.
<br> Inserts a line break within text.
<hr> Creates a horizontal line to separate content.
<strong> Defines strong, emphasized text with bold styling.
<em> Defines text with emphasis, usually styled as italicized text.
<ul> and <ol> <ul> creates an unordered list, and <ol> creates an ordered list.
<li> Defines a list item within <ul> or <ol>.
<a> Defines a hyperlink, using the href attribute to specify the target URL.
href (attribute) Specifies the URL for a hyperlink.
target (attribute) Specifies where to open the linked document (e.g., "_blank" opens it in a new tab).
<img> Embeds an image using attributes like src (source) and alt (alternative text).
src (attribute) Specifies the source of the image file.
alt (attribute) Provides alternative text if the image cannot be displayed.
width (attribute) Specifies the width of an image or element.
height (attribute) Specifies the height of an image or element.
<table> Defines a table in an HTML document.
<tr> Defines a row in a table.
<td> Defines a cell in a table.
<th> Defines a header cell in a table.
colspan (attribute) Specifies the number of columns a cell should span.
rowspan (attribute) Specifies the number of rows a cell should span.
<form> Defines an HTML form for user input.
<input> Defines an input field where users can enter data.
<textarea> Defines a larger text input area for user input.
<label> Defines a label for an input element.
<button> Defines a clickable button.
<audio> Embeds audio content in the document with playback controls.
control (attribute) Specifies that audio or video controls (play, pause, volume) should be displayed.
<meta> Provides metadata like character set and viewport settings.
charset (attribute) Defines the character encoding (e.g., UTF-8).
name (attribute) Specifies the name of a metadata property (e.g., "viewport").
content (attribute) Specifies the value of the metadata property (e.g., "width=device-width").
viewport (attribute) Optimizes the layout for different screen sizes and devices.
