CSS Selectors, Properties, and Concepts

Concept/Property Description
Element Selector Targets HTML elements by their tag name (e.g., p, h1).
Class Selector Targets elements with a specific class, prefixed with a dot (e.g., .example).
ID Selector Targets an element with a specific ID, prefixed with a hash (#).
Attribute Selector Targets elements with a specific attribute or value (e.g., [type="text"]).
Pseudo-class Applies styles based on the state of an element (e.g., :hover).
Pseudo-element Applies styles to specific parts of an element (e.g., ::before, ::after).
Margin Defines the space outside an element's border.
Border Defines the edge of an element.
Padding Defines the space between the content and the border.
Content The actual content inside the padding.
Static The default positioning of elements, not affected by top, left, etc.
Relative Positions an element relative to its normal position.
Absolute Positions an element relative to the nearest positioned ancestor.
Fixed Positions an element relative to the viewport, remaining in place during scrolling.
Sticky Switches between relative and fixed positioning based on the scroll position.
Block Renders elements as a block (full-width) element.
Inline Renders elements inline with no line breaks.
Inline-block Combines inline and block behavior.
None Hides an element from the layout.
Flex Defines a flexible container for layout using flexbox.
Grid Defines a grid container for layout using the CSS grid model.
Justify-content Aligns flex items along the main axis.
Align-items Aligns flex items along the cross axis.
Flex-direction Defines the direction of the flex container's main axis.
Flex-wrap Specifies whether flex items should wrap onto multiple lines.
Grid-template-columns Defines column structure in a grid container.
Grid-template-rows Defines row structure in a grid container.
Gap Defines space between rows and columns in a grid or flex container.
Grid-area Specifies grid items and their placement in the grid.
Font-family Specifies the font of the text.
Font-size Specifies the size of the font.
Font-weight Specifies the thickness of the font (e.g., bold).
Line-height Defines the height of a line of text.
Text-align Defines the horizontal alignment of text.
Text-decoration Applies decoration to text (e.g., underline).
Color Sets the text color.
Background-color Sets the background color of an element.
Opacity Defines the transparency level of an element.
RGBA Defines colors with red, green, blue, and alpha transparency values.
HEX Defines colors using hexadecimal values.
HSL Defines colors using hue, saturation, and lightness values.
PX Specifies absolute units in pixels.
EM Specifies relative units based on the font size of the parent.
REM Specifies relative units based on the root element's font size.
% Specifies values as a percentage of the parent element.
VH Specifies relative units based on the viewport height.
VW Specifies relative units based on the viewport width.
Transition Specifies the transition effect between states.
Transform Applies 2D or 3D transformations to an element (e.g., rotate, scale).
Animation Defines animation keyframes and behavior.
Media Queries Applies styles based on conditions like screen size or device type.
@media Specifies the start of a media query block.
Max-width Defines the maximum width for an element or media query.
Min-width Defines the minimum width for an element or media query.
